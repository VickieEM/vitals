Vitals Checker Web App

How to run:
1. Install Flask:
   pip install flask

2. Initialize the database:
   python init_db.py

3. Run the app:
   python app.py

4. Open in your browser:
   http://127.0.0.1:5000/

Features:
- Vitals input (BP, HR, Temp, Pain Scale)
- Symptoms list (Chest Pain, Cough, Fever, etc.)
- Custom symptoms field
- Preexisting conditions and family history
- Risk assessment engine
- Health tips for normal results
- Admin dashboard to view submissions
- Bootstrap 5 styling
